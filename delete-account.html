Delete Account

So löschst du dein Konto / How to delete your account

  • In der App: Öffne Vibloo → Profil → Einstellungen → Konto löschen
  • In the app: Open Vibloo → Profile → Settings → Delete Account

Was wird gelöscht / What gets deleted

  • Profildaten (Name, Fotos, Bio) / Profile data (name, photos, bio)
  • Chat-Nachrichten / Chat messages
  • Spielstände / Game progress
  • Kaufhistorie / Purchase history
  • Alle gespeicherten Daten / All stored data

Die Löschung erfolgt innerhalb von 30 Tagen. / Deletion is completed within 30 days.

Per E-Mail / By email

Du kannst die Löschung auch per E-Mail anfordern. / You can also request deletion by email.
